# Skill: Cloud Backend

用于推进微信云开发的云函数、数据库集合、权限和业务计算。

## 使用场景

- 设计云数据库集合
- 编写云函数
- 实现登录态、用户权限
- 实现热量、营养素、统计、打卡计算

## 执行步骤

1. 根据当前阶段 spec 建立集合和字段
2. 云函数按业务域拆分，避免一个函数包揽所有逻辑
3. 所有写接口校验登录态
4. 所有用户私有数据按用户身份过滤
5. 所有核心计算放服务端或服务端可复算
6. 小程序端只展示结果，不信任前端传入的用户身份

## 建议云函数

- `auth`
- `profile`
- `goal`
- `food`
- `diet`
- `exercise`
- `body`
- `stats`
- `checkin`
- `feedback`
- `admin`

## 安全要求

- 不向前端返回 `session_key`
- 不在前端保存密钥
- 管理后台接口必须校验管理员身份
- 删除和禁用类操作必须记录操作时间
- 用户只能访问自己的健康、饮食、运动、体重数据
