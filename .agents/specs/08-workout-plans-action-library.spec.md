# Phase 8 Spec: 训练计划与动作库

## 目标

补齐健身侧内容，让用户不仅能记录运动，还能按目标执行基础训练计划。

## 明确不做

- 付费课程
- 会员专属训练
- 教练咨询
- 直播课
- 复杂私教排课
- 微信支付

## 页面范围

- 动作库列表页
- 动作详情页
- 训练计划列表页
- 训练计划详情页
- 今日训练页
- 管理后台动作管理页
- 管理后台训练计划管理页

## 功能 1: 动作库

动作分类：

- 胸部
- 背部
- 肩部
- 手臂
- 腹部
- 臀腿
- 全身
- 有氧
- 拉伸

动作字段：

- 动作名称
- 封面，可选
- 动作视频或动图，可选
- 目标肌群
- 辅助肌群，可选
- 难度
- 所需器械
- 动作步骤
- 常见错误
- 注意事项

验收：

- 可按分类查看动作
- 可搜索动作
- 动作详情信息完整

## 功能 2: 训练计划

计划类型：

- 减脂
- 增肌
- 塑形
- 新手入门
- 居家训练
- 健身房训练

计划字段：

- 计划名称
- 封面，可选
- 目标
- 难度
- 周期
- 每周次数
- 单次预计时长
- 动作列表
- 每个动作组数
- 每组次数
- 休息时间

验收：

- 可查看训练计划列表
- 可查看计划详情
- 可选择一个计划作为当前计划

## 功能 3: 今日训练

规则：

- 根据用户当前计划展示今日动作
- 用户可逐项标记完成
- 支持填写实际次数和重量，可选
- 支持休息倒计时
- 完成后生成运动记录

验收：

- 今日训练可开始
- 动作可逐项完成
- 完成后写入运动记录
- 首页运动消耗同步变化

## 功能 4: 后台训练内容管理

支持：

- 新增动作
- 编辑动作
- 上架动作
- 下架动作
- 新增训练计划
- 编辑训练计划
- 上架训练计划
- 下架训练计划

验收：

- 后台动作和计划能同步到小程序端
- 下架内容小程序端不可见

## 数据建议

新增集合：

- `exercise_actions`
- `workout_plans`
- `workout_plan_actions`
- `user_workout_plans`
- `workout_sessions`

## 验收标准

- 动作库可浏览和搜索
- 训练计划可浏览、选择、执行
- 今日训练完成后能生成运动记录
- 不出现会员专属、付费课程、支付入口
