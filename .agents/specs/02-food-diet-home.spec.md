# Phase 2 Spec: 食材库、饮食记录、首页仪表盘

## 目标

完成 MVP 最核心的饮食记录闭环：搜索食材、添加饮食、计算热量、首页实时反馈。

## 页面范围

- 首页
- 今日饮食详情页
- 添加饮食页
- 食物搜索页
- 自定义食物页
- 我的自定义食物页

## 食材库功能

支持：

- 食材分类
- 食材搜索
- 平台食材
- 用户自定义食物

搜索支持：

- 食物名称关键词
- 分类筛选

第一版不做：

- 拼音搜索
- 条形码搜索
- 拍照识别
- 用户自定义食物公开审核

## 饮食记录功能

餐次：

- 早餐
- 午餐
- 晚餐
- 加餐

添加方式：

- 从食材库搜索添加
- 从自定义食物添加
- 手动创建自定义食物后添加

饮食记录字段：

- 日期
- 餐次
- 食物名称
- 重量 g
- 热量
- 蛋白质
- 碳水
- 脂肪
- 备注，可选

## 计算规则

```text
总热量 = 每100g热量 * 重量 / 100
总蛋白质 = 每100g蛋白质 * 重量 / 100
总碳水 = 每100g碳水 * 重量 / 100
总脂肪 = 每100g脂肪 * 重量 / 100
```

重量支持一位小数。

## 首页仪表盘

首页展示：

- 今日推荐摄入热量
- 今日已摄入热量
- 今日剩余热量
- 今日运动消耗热量，Phase 2 可先显示 0
- 蛋白质进度
- 碳水进度
- 脂肪进度
- 早餐、午餐、晚餐、加餐入口
- 今日饮食打卡状态

## 复制昨日饮食

第一版支持复制昨日饮食：

- 将昨日全部饮食记录复制到今日
- 复制后可编辑或删除
- 不复制创建时间，只复制食物、餐次、重量和营养数据

## 验收标准

- 可新增平台食材
- 可搜索食材
- 可新增自定义食物
- 可将食物加入指定餐次
- 修改重量后热量和营养素自动变化
- 可编辑饮食记录
- 可删除饮食记录
- 首页摄入热量随记录实时变化
- 首页营养素进度随记录实时变化
- 添加饮食后自动完成饮食打卡
- 可复制昨日饮食
