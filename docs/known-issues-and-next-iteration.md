# 已知问题与二期建议

## 已知问题

- 真实微信登录、真机体验、云函数部署状态、云数据库权限依赖微信开发者工具和云环境，无法仅通过本地命令完全验证。
- 管理后台为小程序内基础后台，适合 MVP 维护；复杂运营场景建议二期建设独立 Web 后台。
- 管理员账号需要上线前在 `admin_users` 手动配置；未配置时后台无法登录。
- 食材历史数据如果云端已重复，本版本会前后端展示去重，但不会物理清理历史重复文档。

## 二期建议

- 独立 Web 管理后台和更细粒度管理员权限。
- 食材批量导入、去重清洗和审核流程。
- 自动化端到端测试和云函数单元测试。
- 更细的营养素图表、周/月趋势和目标达成分析。
- 消息提醒、数据导出、可穿戴设备同步等扩展能力。
