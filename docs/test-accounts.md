# 测试账号清单

微信小程序测试账号需要使用真实微信或开发者工具模拟用户登录。以下为上线前验收所需账号准备模板。

| 账号类型 | 准备方式 | 验收用途 |
| --- | --- | --- |
| 新用户 | 使用未在当前云环境登录过的微信进入小程序 | 登录、建档、目标设置、空状态 |
| 已建档普通用户 | 完成健康档案和目标设置，至少添加一条饮食记录 | 老用户再次进入、首页数据、修改档案 |
| 7 天记录用户 | 连续准备近 7 天饮食、运动、体重记录 | 数据趋势、连续打卡、统计页 |
| 管理员账号 | 在 `admin_users` 手动写入项目管理员 openid，`status` 设为 `active` | 后台概览、食材维护、反馈处理 |
| 被禁用用户 | 在后台用户管理中将普通用户禁用 | 登录拒绝、受限接口校验 |

正式验收时请记录每个测试微信对应的昵称、账号状态、准备人。openid 只在内部验收表中记录，不在用户侧展示。
